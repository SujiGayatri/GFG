<h2><a href="https://www.geeksforgeeks.org/problems/count-element-occurences/1?page=1&category=Arrays&difficulty=Easy&status=unsolved&sortBy=submissions">Count Frequent Elements</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given an array <strong>arr[]</strong> of <strong>n</strong> integers and an integer <strong>k</strong>, find the count of elements that appear more than <strong>n/k</strong> times in the array.</span></p>
<p><span style="font-size: 18px;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>arr[] = [3, 1, 2, 2, 1, 2, 3, 3], k = 4
<strong>Output: </strong>2<strong>
Explanation: </strong>The elements 2 and 3 each occur 3 times, which is more than n/k = 2.</span>
</pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>arr = [2, 3, 3, 2], k = 3
<strong>Output: </strong>2<strong>
Explanation: </strong>Both 2 and 3 appear 2 times in the array, which is more than n/k.</span></pre>
